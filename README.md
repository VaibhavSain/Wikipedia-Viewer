# 🌐 Wikipedia Viewer

A simple and interactive **Wikipedia Viewer** built using **HTML, CSS, and JavaScript**.  
This application allows users to search for any topic and instantly view summaries from Wikipedia using the Wikipedia REST API.

🔗 **Live Demo:**  
https://zingy-selkie-449d19.netlify.app/

💻 **GitHub Repository:**  
https://github.com/VaibhavSain/Wikipedia-Viewer

---

## 🚀 Features

- 🔎 Search for any topic on Wikipedia
- 📄 Display short summaries of articles
- ⚡ Fast and dynamic results using JavaScript
- 🎨 Clean and responsive UI
- 🌍 Fetches real-time data from the Wikipedia API

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **Wikipedia REST API**

## 📸 How It Works

1. The user enters a topic in the search bar.
2. JavaScript sends a request to the **Wikipedia API**.
3. The API returns a summary and description of the topic.
4. The result is dynamically displayed on the page.
